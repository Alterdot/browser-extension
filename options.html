Alterdot Domain Access Identifiers
  Whenever you try to access an Alterdot domain our extension looks for one of several special identifiers that let it know that you are indeed looking for an Alterdot domain. There are three default identifiers, .adot and .a which are added at the end of the domain name as well as a. which is added in front. For example, by searching for either bitcoin.adot, bitcoin.a or a.bitcoin you will be taken to the "bitcoin" domain registered on our BlockchainDNS. If you want to access a normal domain with the name "a", e.g. "a.com", disable Intercepted Search in the main extension popup.

  Important! If Intercepted Search is disabled you have to add "/" at the end of the domain request like bitcoin.adot/ or bitcoin.a/. Beware that a.bitcoin/ format is not supported.
